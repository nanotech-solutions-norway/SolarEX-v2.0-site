# SolarEX v2.0 Site Repository Template

This package is a static-first SolarEX v2.0 site template generated from the April 9, 2026 website blueprint and implementation reports.

## Recommended repository name
- nanotech-solutions-norway/SolarEX-v2.0-site

## Included
- Proof-led homepage
- Solutions overview
- Quartz and Titan product pages
- Use-case fit page
- Proof stack page
- Resources hub
- Case studies hub and Scandinavian case page
- Technical specifications
- ROI analysis
- Application process
- About, Contact, Privacy, Accessibility
- Static logo and favicon assets
- `v2.0` version badge in header
- `robots.txt` and `sitemap.xml`

## Immediate actions after repository creation
1. Create a new public repository in GitHub named `SolarEX-v2.0-site`.
2. Upload the contents of this folder to the repository root.
3. Enable GitHub Pages or deploy through the preferred publishing path.
4. Replace privacy and accessibility placeholders with approved text.
5. Add final production media assets if the current SVG placeholders are to be replaced.
6. Verify all download links and contact routing.
