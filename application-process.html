Application process

From site intake to treated-versus-control pilot review.

The reports recommended process pages that reduce uncertainty and make next steps operationally clear. SolarEX v2.0 therefore defines a practical evaluation flow rather than relying on generic “contact us” language only.

1. Intake

Collect site location, installed capacity, panel type, fouling profile, and current wash model.

2. Mechanism match

Determine whether Quartz, Titan, or a mixed review is technically justified.

3. Pilot design

Define treated and control zones, monitoring period, and key evidence outputs.

4. Evaluation

Translate the result into yield, water, labor, and O&M economics.

Inputs that improve the recommendation

  • Representative site photos
  • Cleaning history and current interval
  • Fouling type and environmental conditions
  • Electricity price or commercial value basis

Outputs the pilot should produce

  • Treated-versus-control production comparison
  • Visual cleanliness retention evidence
  • Wash reduction and water-use data
  • Site-specific ROI interpretation
